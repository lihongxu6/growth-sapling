v2.0 开屏页（Splash）设计稿说明

  • 定位：独立小程序页面 pages/splash/splash，在 app.json 中注册为入口页；由 app.js onLaunch 或启动逻辑自动进入。
  • 视觉：全屏居中，背景色 --bg #FFF8F0；沿用 H5 开屏设计：🌱 logo + 产品名 + slogan。
  • 动画：logo 由 0.6 缩放淡入（splashPop，600ms ease），文字依次错开 150ms/250ms 出现；小程序中保持同动画曲线。
  • 停留：展示约 1.5s 后，整页 opacity 0 淡出（600ms transition），然后 wx.reLaunch 或 wx.redirectTo 到打卡页。
  • 跳过：点击屏幕任意区域可立即淡出并跳转；pointer-events 在淡出过程中禁用。不显示底部提示语，避免分散用户注意力。
  • 状态栏：按 design-spec §3.1，仅保留 44px 安全区，不显示品牌名/假时间/信号/电量。
  • 与 H5 差异：H5 中 splash 是 index.html 内的 overlay；小程序中为独立页面，跳转后不存在回退问题。
