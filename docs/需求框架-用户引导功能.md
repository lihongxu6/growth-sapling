> 关联：受众=儿童、#57（相册/摄像头隐私声明 + 授权弹窗）、§3.8（不堆独立页，用单页操作手册）、§3.10（不动物上线内容）、§3.15、§3.20。

---

## 1. 背景

- v2.0 本地版（开屏页 + 成长页用户信息区）已提交审核（2.0.1）。
- 缺口：儿童首次进入不知道从哪开始、数据去哪了、为何要相册/摄像头授权。
- §3.15 决策：首次引导 + 隐私透明提示统一放到下一版，从界面设计整体做。

---

## 2. 目标

- **G-目标1**：儿童首次进入，通过「跟着做一遍」掌握核心操作（建任务 / 打卡 / 补卡 / 看日历 / 看徽章 / 改头像昵称）。
- **G-目标2**：引导中透明说明「数据仅存本机、不上传」，调用相册/摄像头前弹合规授权。
- **G-目标3**：引导所建任务沉淀为真实账户数据（不删）——引导即首次真实使用。
- **G-目标4**：成长页提供常驻「操作手册」入口，随时回看。
- **G-目标5（情感连接）**：以**松鼠果果（产品 IP）拟人化**贯穿引导与操作手册，建立孩子与产品的情感连接。

---

## 3. 用户与场景

- **主用户**：7–8 岁儿童（**受众=儿童**，文案童趣/图文，家长可协助）。
- **场景 A（冷启动·首次引导）**：清空数据 → 松鼠引导儿童走完整核心流程。
- **场景 B（授权疑虑）**：改头像弹相册/摄像头授权，引导已提前说明。
- **场景 C（随时回看）**：使用中忘记某功能，从成长页操作手册图标看图文说明。

| 编号 | 功能 | 说明 | 优先级 |
|---|---|---|---|
| **G1** | 交互式首次引导（弱引导 / 可跳过）| 清空数据 → 松鼠分步高亮真实界面，引导儿童完成 创建任务 → 打卡 → 补卡 → 看日历 → 看徽章 → 改头像昵称；所建任务沉淀真实账户，不删 | P0 |

## 5. 范围外

- 账户/云同步引导（已推迟 §3.14）
- 退出登录（本地版无）
- 多页独立「教程中心」（用单页操作手册替代，§3.8）
- 「重新走一遍交互引导」按钮（用户选纯静态图文，不做）

---

## 6. 关键规则（已确认）

- **R1 清空保护**：清空仅「真正首次（本地无任务数据且从未引导过）」时执行一次；中途退出保留进度（不重复清空、不回滚）；操作手册绝不触发清空。
- **R2 引导即真实使用**：引导所建任务 = 真实账户任务，走完即首个真实数据，不删。
- **R3 受众童趣**：文案儿童语言 + 图文/emoji。
- **R4 松鼠拟人**：引导角色 = 松鼠果果，以拟人化形象（对话气泡/动作）贯穿引导与操作手册，建立情感连接；当前用占位形象（emoji/默认松鼠），真实 IP 到位替换（§1）。
- **R5 弱引导 + 可跳过**：仅高亮提示 + 松鼠气泡，不强制顺序、不打断；提供「跳过」直接进首页。
